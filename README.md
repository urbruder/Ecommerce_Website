
# 🛍️ ShopKart – E-commerce Website

ShopKart is a MERN stack-based e-commerce web application that provides a seamless shopping experience with product browsing, cart management, authentication, and multiple payment options including Cash on Delivery (COD) and Stripe.
   
It also includes a dedicated Admin Panel for managing products, checking orders, and handling inventory.
Admin Panel: https://shopkart-admin.vercel.app/

Backend: https://shopkart-backend-qur5.onrender.com
ShopKart website: https://ecommerce-website-lake-three-10.vercel.app/

## Features
### User Side

- 🏠 Homepage with Latest Arrivals

- 👕 Product Collections with Filters (Men, Women, Kids, Categories & Types)

- 🔍 Search & Sort functionality

- 🛒 Add to Cart, Update Quantity, Remove Items

- 👤 User Authentication (Login & Signup)

- 💳 Payment Methods:

  - Stripe (Online Payment)

  - Cash on Delivery (COD)

- 📦 Order Checkout with Delivery Information Form

- 📱 Responsive UI for all devices

### Admin Side

- 📋 List all products

- ➕ Add new products (with images, price, stock, category)

- 🛍️ Manage existing products (Edit / Delete)

- 📦 View and Manage Orders (status update, track orders)

- 🔐 Secure Admin Authentication



## Tech Stack

**Frontend:** React.js, Redux, TailwindCSS / CSS

**Backend:** Node.js, Express.js

**Database:** MongoDB

**Authentication:** JWT (JSON Web Tokens)

**Payment Integration:** Stripe

**Hosting:** Vercel , MongoDB Atlas (Database)
